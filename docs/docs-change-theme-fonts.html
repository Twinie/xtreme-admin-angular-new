How to Change Global Fonts ?

we have used google fonts in our template. so, when you change the fonts of the theme go tho the scss folder and open variable.scss file. In that we have import the google fonts link.
Note:
First run the demo you have and if you don't know how to run then follow this link project 
                    
ng serve -o
1) Go to the src assets scss variable.scss . 
                                    
@import url('https://fonts.googleapis.com/css2?family=DM+Sans:wght@400;500;700');
                                    
                                  
                                    
$font-family-sans-serif: 'Nunito Sans',sans-serif !default;
2) src index.html 
                          
<link
href="https://fonts.googleapis.com/css2?family=DM+Sans:wght@400;500;700&display=swap"
rel="stylesheet"
/>
Check in the browser after saving the variable.scss and index.html[src].
If you want to change fonts of the theme then put the URL of your fonts and change font family in variable.scss.
